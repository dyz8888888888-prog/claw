## 一、概念板块龙头（龙一/龙二/龙三...龙十）

### 1.1 龙头是如何定义的

**核心结论：龙头不是交易所官宣的，是资金「实时投票」赛出来的。**

量化判定四步法（行业共识）：

| 步骤 | 动作 | 量化标准 |
|------|------|----------|
| 1. 划定范围 | 筛选走强板块 | 板块涨幅≥3% 且 板块内≥3只涨停 |
| 2. 三维打分 | 给每只涨停股打分 | ① 封板时间分（越早分越高） ② 封单资金分（封单/流通市值） ③ 连板高度分 |
| 3. 实时刷新 | 动态排序 | 炸板/回封时3秒内重算，榜单即时切换 |
| 4. 收盘锁定 | 最终排名 | 收盘封死未炸板者，总分排前N即为龙N |

**排序权重建议：**
```
涨停时间: 10:00前=100分, 10:30前=90分, 11:00前=80分, 午后=60分
封单力度: 封单/流通市值>8%=100分, >5%=80分, >2%=60分
连板高度: 连板数×10
```
**同分时谁先封死谁在前。**

| 数据源 | 原生龙头接口 | 涨停时间 | 封单额 | 连板数 | 可自算龙头 |
|--------|:----------:|:------:|:-----:|:-----:|:---------:|
| **kaipanla-crawler** (免费) | ❌ 无原生排序 | ✅ | ✅ | ✅ | ✅ 满分 |
| **Tushare kpl_list** (需5k积分) | ❌ | ✅ `lu_time` | ✅ `limit_order` | ✅ `status` | ✅ 可算 |
| **开盘啦付费API** (¥440买断) | ✅ 原生龙头排序 | ✅ | ✅ | ✅ | ✅ 最方便 |
| **AKShare** (免费) | ❌ | ✅ | ✅ `封板资金` | ❌ 无 | ⚠️ 缺连板 |
| **StockAPI** (付费) | ❌ | ❌ | ❌ | ❌ | ❌ 无涨停原因 |

### 1.3 最优方案

**首选：kaipanla-crawler（免费，GitHub MIT协议）**
- `get_limit_up_sectors()` 返回每个概念的涨停股列表，含涨停时间、封单额、连板天数
- 只需在此基础上实现四步排序算法，即可输出龙一~龙N

```python
# 伪代码：从 kaipanla-crawler 数据推算龙一龙二
for sector in data['sectors']:
    stocks = sector['stocks']
    for s in stocks:
        time_score = calc_time_score(s['涨停时间'])      # 越早越高
        fengdan_score = calc_fengdan_score(s['封单额'], s['流通市值'])  # 封单/流通市值
        lianban_score = s['连板天数'] * 10
        s['dragon_score'] = time_score + fengdan_score + lianban_score
    sector['dragons'] = sorted(stocks, key=lambda x: x['dragon_score'], reverse=True)
```

## 二、涨停原因 & 概念标签

### 2.1 覆盖率对比

| 字段 | kaipanla-crawler | Tushare kpl_list | AKShare | StockAPI |
|------|:---:|:---:|:---:|:---:|
| **涨停原因** | ✅ 详细文本（如"芯片(先进封装)；美国白宫..."） | ✅ `lu_desc` | ❌ | ❌ |
| **概念标签** | ✅ `概念标签`（如"先进封装、存储"） | ⚠️ `theme`（有限，无逗号分隔多概念） | ❌ | ❌ |
| **涨停主题** | ✅ `主题`（如"先进封装"） | ⚠️ `theme` | ❌ | ❌ |
| **连板描述** | ✅ `连板描述`（如"2连板"） | ✅ `status` | ❌ | ❌ |

### 2.2 结论

**kaipanla-crawler 在涨停原因和概念标签上是最全面的免费方案。**
- 每个涨停股都有详细的「涨停原因」字段，包含具体事件驱动原因
- 「概念标签」字段是逗号分隔的多概念标签
- 「主题」字段是主概念

## 三、市场情绪指标体系

### 3.1 核心情绪指标全景

```
┌─────────────────────────────────────────────────────────┐
│                    市场情绪量化六维                       │
├───────────────┬─────────────────┬──────────────────────────┤
│ 维度          │ 核心指标         │ 计算/来源                │
├───────────────┼─────────────────┼──────────────────────────┤
│ 1.涨停强度    │ 涨停家数         │ 当日封死涨停的股票总数    │
│               │ 跌停家数         │ 当日封死跌停的股票总数    │
│               │ 涨停溢价率       │ 昨日涨停股今日平均涨幅    │
├───────────────┼─────────────────┼──────────────────────────┤
│ 2.打板质量    │ 炸板率           │ 炸板数/(涨停+炸板)       │
│               │ 封板成功率       │ 1-炸板率                  │
│               │ 盘中破板率       │ 破板后回封/涨停总数       │
├───────────────┼─────────────────┼──────────────────────────┤
│ 3.接力情绪    │ 连板晋级率       │ 今日连板/昨日涨停         │
│               │ 二板晋级率       │ 今日≥2连板/昨日首板       │
│               │ 高标存活率       │ 5板+高标涨停/昨日5板+总数 │
├───────────────┼─────────────────┼──────────────────────────┤
│ 4.亏钱效应    │ 大面股数         │ 打板浮亏>8%个股数         │
│               │ 昨日曾涨停表现   │ 昨炸板股今日平均涨幅      │
│               │ 天地板数         │ 涨停→跌停个股数           │
├───────────────┼─────────────────┼──────────────────────────┤
│ 5.赚钱效应    │ 昨日涨停表现     │ 昨日涨停股今日均收益      │
│               │ 昨日连板表现     │ 昨日≥2连板股今日均收益    │
│               │ 百日新高数       │ 创100日新高个股数         │
├───────────────┼─────────────────┼──────────────────────────┤
│ 6.市场广度    │ 涨跌比           │ 上涨家数/下跌家数         │
│               │ 成交额(全市场)   │ 两市成交总额              │
│               │ 多头/空头风向标  │ 最强/最弱3只股            │
└───────────────┴─────────────────┴──────────────────────────┘
```

|--------|------|------|------|------|

### 4.2 东方财富裸接口（推荐免费方案）

概念板块资金流向排行页：`https://push2his.eastmoney.com/api/qt/clist/get`
- 参数 `fs=m:90+t:3` (概念板块)，`fid=f62` (按主力净流入排序)
- 返回：板块代码/名称/涨跌幅/主力净流入/超大单净流入/大单净流入 等

```

  ?pn=1&pz=50&fs=m:90+t:3&fid=f62&fields=f2,f3,f4,f12,f14,f62,f184,f66
```
