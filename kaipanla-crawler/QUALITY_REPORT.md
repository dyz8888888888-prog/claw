## 已知问题

1. **异动个股接口不稳定**: 实时异动个股有时返回空JSON导致解析失败
2. **实际涨跌停计数为0**: `get_realtime_actual_limit_up_down` 返回的 `actual_limit_up` 和 `limit_up` 均为0，但市场情绪返回涨停94家，需交叉验证
3. **历史数据可用性**: 最近交易日(如0619周五)部分数据可能延迟上传
4. **历史连板梯队详情**: `get_consecutive_limit_up` 对某些历史日期返回空(需逐板查询)

## 适用场景

| 场景 | 推荐接口 | 理由 |
|------|----------|------|
| 市场情绪监控 | `get_realtime_market_mood` | 秒级响应，涵盖涨跌停和涨跌比 |
| 板块热度排行 | `get_sector_ranking` | 涨停原因+板块+个股详情 |
| 连板梯队追踪 | `get_market_limit_up_ladder` | 全市场连板+反包板+高度标注 |
| 板块资金流 | `get_sector_capital_data` | 成交额+主力净额+涨跌幅 |
| 情绪周期判断 | `get_realtime_rise_fall_analysis` | 炸板率+昨日打板表现 |
| 回溯研究 | `get_daily_data` + `get_new_high_data` | 多日历史数据比对 |

## 与可转债作战台集成建议

开盘啦数据可作为情绪辅助层，为 cb_monitor 提供:
- **市场温度**: 涨跌比、涨停数 → 决策权重调整
- **主线追踪**: 涨停板块排行 → 概念映射到可转债
- **风险预警**: 炸板率、大幅回撤 → 风控信号
- **周期定位**: 百日新高趋势 → 市场阶段判断
